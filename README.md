# eBPF Study Repository

이 레포지토리는 eBPF를 공부하기 위한 목적으로 만들어졌습니다. eBPF는 리눅스 커널에서 실행되는 고성능 확장 가능한 프로그램을 작성할 수 있는 강력한 기술입니다. 이 프로젝트는 파일 및 네트워크 리소스 추적과 같은 기능을 제공하는 eBPF 프로그램을 개발하는 데 중점을 둡니다.

## 주요 파일

- `file.bpf.c`: 파일 관련 이벤트를 추적하는 eBPF 프로그램.
- `network.bpf.c`: 네트워크 트래픽을 모니터링하는 eBPF 프로그램.
- `process.bpf.c`: 프로세스 관련 이벤트를 추적하는 eBPF 프로그램.
- `loader.c`: eBPF 프로그램을 로드하고 관리하는 유저 스페이스 코드.
- `Makefile`: 빌드 및 설치를 위한 Makefile.

## 포함된 헤더 파일

- `include/bpf.h`: eBPF 프로그램에서 사용되는 공통 헤더 파일.
- `include/common.h`: 프로젝트 전반에서 사용되는 공통 정의.
- `include/map.h`: eBPF 맵 관련 유틸리티.
- `include/vmlinux.h`: 리눅스 커널 구조체 정의.

## 사용 방법

1. 프로젝트를 빌드합니다:
   ```bash
   make
   ```

2. eBPF 프로그램을 로드합니다:
   ```bash
   ./loader
   ```

3. 필요한 기능을 테스트하거나 확장합니다.
